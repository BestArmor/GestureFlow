# 🖱️ GestureFlow

Control your PC with a flick of your wrist 💜

## ✨ Features
- 🎨 Neon purple trail
- ⚡ < 100ms latency
- 🎯 Smart click detection
- 🖥️ Multi-monitor support

## 🎬 Gestures
| Gesture | Action |
|:---:|---|
| ← | Browser Back |
| → | Browser Forward |
| ↑ | New Tab |
| ↓ | Close Tab |
